Bitkit (https://www.bitkit.to) är en enkel, men kraftfull Wallet med självförvaltare. Betala vem som helst, var som helst, när som helst.


Bitkit är en självförvaltande mobil Wallet som ger dig möjlighet att ta verklig Ownership av din Bitcoin så att du kan spendera på dina villkor. Förbättrad med utmärkande funktioner och en snygg design låser Bitkit upp omedelbara betalningar till vem som helst, när som helst, var som helst. Allt samtidigt som det är helt öppen källkod för alla att granska.

## guide


Bitkit är verkligen mycket enkelt att använda.



Som en fullfjädrad Bitcoin Wallet innehåller Bitkit alla de funktioner du förväntar dig:


Omedelbara betalningar: Inget mer jonglerande mellan plånböcker för On-Chain och Lightning-transaktioner. Bitkit blandar dem båda sömlöst.


Hantering av saldo: Överför enkelt pengar mellan ditt sparkonto och ditt utgiftskonto så att du alltid har tillräckligt med kapacitet för omedelbara betalningar.


Återställningsfras: Återställ ditt sparsaldo på alla Wallet som stöder BIP 39.


Automatisk säkerhetskopiering: Icke-känsliga data från din Wallet säkerhetskopieras automatiskt så att du alltid kan återställa ditt utgiftssaldo.


Detaljerad transaktionshistorik: Tilldela kontakter och tagga dina transaktioner för att hålla dem organiserade.


![cover](assets/1.webp)


Bitkit har också unika funktioner som skiljer det från mängden:


Betalbara kontakter: Säg adjö till att be om adresser eller fakturor. Lägg bara till vänner i din kontaktlista och betala dem.


Live widgets: Gör din Wallet-startskärm lite roligare och mer användbar med engagerande widgetar.


Social profil: Ta kontroll över din offentliga profil och dina länkar, så att dina kontakter kan kontakta dig och betala dig när som helst.


Lösenordslösa konton: Logga in på webbplatser som stöder Slashtags eller Lightning-autentisering.


![cover](assets/2.webp)


För experter erbjuder Bitkit kraftfulla alternativ:


Anpassad avgift: Välj din nätverksavgift och öka transaktionerna för snabbare bekräftelse.


Externa blixtanslutningar: Få ditt nod-ID så att du kan få anslutningar från vilken peer som helst.


Kontroll över mynt: Välj vilka mynt som ska användas för varje transaktion.


Electrum server: Synkronisera med Blockchain via den server du föredrar.


Address Visare: Visa mottagnings- och ändringsadresser som härrör från din seed.


Address typ: Ta emot betalningar via Legacy-, Nested SegWit- eller Native SegWit-adresser.


![cover](assets/3.webp)


Köpa eller sälja Bitcoin

Bitkit stöder inte köp och försäljning av Bitcoin. För att köpa eller sälja, använd börser som Bitfinex och skicka sedan till eller från Bitkit.
