## BlueWallet: en introduktion


![image](assets/1.webp)


För att kunna ta emot eller skicka Bitcoin från din mobiltelefon (eller surfplatta), ladda ner BlueWallet från motsvarande webbplats för din typ av mobiltelefon eller surfplatta:



- iPhone: https://itunes.apple.com/app/bluewallet-Bitcoin-Wallet/id1376878040
- Android: https://play.google.com/store/apps/details?id=io.bluewallet.bluewallet


När programmet har installerats skapar du din första Wallet:


1. Wallet namn: Min Bitcoin (till exempel)

2. Typ: Bitcoin

3. Skapa


_⚠️ Anteckna säkerhetskopieringsfrasen (12 ord) på ett papper. Om du tappar bort din telefon eller raderar BlueWallet-appen är detta det enda sättet att återställa dina Bitcoin! Kontrollera, dubbelkolla och ta ALDRIG bilder av papperet, visa det eller dela det med någon! Förvara det på en säker plats utom räckhåll för nyfikna ögon


### Ta emot din första Bitcoin


Från din nya BlueWallet Wallet, använd alternativet "Receive", vilket kommer att generate en unik betalning Address (börjar med bc1...)


![image](assets/2.webp)


1. Låt din mottagande Address skannas direkt med hjälp av QR-koden;

2. ELLER kopiera/klistra in Bitcoin Address;

3. ELLER dela denna Address med hjälp av andra tillgängliga metoder på din enhet.


### Skicka din första Bitcoin


Från din nya BlueWallet Wallet använder du alternativet "Skicka", som gör att du kan välja det belopp som ska skickas och skanna eller klistra in mottagarens Bitcoin Address (börjar vanligtvis med bc1...).


![image](assets/3.webp)


1. Ange det belopp som ska skickas;

2. Skanna mottagarens QR-kod eller klistra in deras Bitcoin Address direkt;

3. Justera transaktionskostnaderna i nätverket genom att sätta ett högre belopp för snabbare bekräftelse;

4. Fortsätt till nästa steg för att bekräfta och skicka din transaktion!


🥇 Den här introduktionen är bara början! Prova det själv och med vänner... BlueWallet erbjuder fler kraftfulla funktioner som du kan utforska i din egen takt.
