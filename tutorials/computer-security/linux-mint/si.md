## Kaj je narobe, če uporabljate običajen računalnik?


Bitcoin ගනුදෙනු කරන විට, ඔබේ පරිගණකය මාලවේئر නොමැති නම් එය ආදර්ශවත් වේ. පැහැදිලියි.


ඔබේ Bitcoin seed වාක්‍යය (සාමාන්‍යයෙන් වචන 12 හෝ 24) අත්සන් කිරීමේ උපාංගයක් (උදාහරණයක් ලෙස Hardware Wallet – එහි ප්‍රධාන අරමුණ) සමඟ පරිගණකයෙන් ඉවත් තබන්නේ නම්, "පිරිසිදු" පරිගණකයක් තිබීම එතරම් වැදගත් නොවේ යැයි ඔබ සිතන්නට පුළුවනි – එය සත්‍ය නොවේ.


මැල්වෙයාර් ආසාදිත පරිගණකයක් ඔබේ Bitcoin ලිපින කියවිය හැකි අතර, ඔබේ ශේෂය ප්‍රහාරකයෙකුට අනාවරණය කරයි - ඔවුන්ට Address දැන ගැනීමෙන් පමණක් Bitcoin ගත නොහැකි වුවත්, ඔබට කොපමණ තිබේදැයි බලන්න, සහ ඔබ වටිනා ඉලක්කයක්ද යන්න ගණනය කරන්න. ඔවුන්ට ඔබ කොතැන ජීවත් වන්නේද යන්න ද කිසියම් ආකාරයකින් සොයා ගත හැකි අතර, උදාහරණයක් ලෙස, ඔබට මුදල් ගෙවීමට කප්පම් ගෙවීමට ඔබගෙන් නියපොතු හෝ දරුවන් ලබා ගත හැකිය.


## ඇතැම් විසඳුම කුමක්ද?


මම බොහෝ බිට්කොයින් භාවිතා කරන්නන්ට Bitcoin ගනුදෙනු කිරීම සඳහා අධිෂ්ඨිත මැල්වෙයාර්-රහිත පරිගණකයක් (අන්තර්ජාල ප්‍රවේශය සහිත) භාවිතා කිරීමට උත්සාහ කරමි. මම පුද්ගලයින්ට Linux Mint වැනි විවෘත මූලාශ්‍ර මෙහෙයුම් පද්ධතියක් භාවිතා කිරීමට යෝජනා කරමි, නමුත් ඔබට අවශ්‍ය නම් Windows හෝ Mac භාවිතා කරන්න – එය නිතරම මැල්වෙයාර් සඟවා ඇති සාමාන්‍ය, හොඳින් භාවිතා කරන ලද පරිගණකයක් භාවිතා කිරීමට වඩා හොඳය.


එක් බාධාවක් නම් එවැනි පරිගණකවල නව මෙහෙයුම් පද්ධතියක් ස්ථාපනය කිරීමයි. එය සදහා මෙම මාර්ගෝපදේශය උපකාරී වේ.


Linux විශේෂාංග බොහොමයක් ඇත සහ මම කිහිපයක් උත්සාහ කර ඇත. මගේ Bitcoiners සඳහා නිර්දේශය වන්නේ Linux Mint, එය ස්ථාපනය කිරීමට පහසු, ඉතා වේගවත් (විශේෂයෙන්ම ආරම්භය සහ වසා දැමීමේදී), අතිරේක මෘදුකාංගයක් නොමැති (සෑම අතිරේක මෘදුකාංග කැබැල්ලක්ම අවදානමක් වේ), සහ මට හෝ වෙනත් අනුවාදවල (Ubuntu සහ Debian වැනි) සසඳන විට අසාමාන්‍ය ලෙස ක්‍රියා නොකළ හෝ අමුතු ලෙස හැසිරී නැත.


සමහරු නව මෙහෙයුම් පද්ධතියකට ඉතා ප්‍රතිරෝධී විය හැක, Windows හෝ Mac OS කැමති විය හැක. මම තේරුම් ගන්නවා, නමුත් Windows සහ Apple මෙහෙයුම් පද්ධති වසා ඇති මූලාශ්‍ර වේ, එබැවින් ඔවුන් කරන දේ විශ්වාස කළ යුතුය; මම හිතන්නේ එය හොඳ ප්‍රතිපත්තියක් නොවේ, නමුත් එය සම්පූර්ණයෙන්ම හෝ කිසිවක් නොවේ. මම කැමති වන්නේ මිනිසුන්ට හොඳින් භාවිතා කරන ලද පරිගණකයකට වඩා නවින්තම Windows හෝ Mac OS පරිගණකයක් භාවිතා කිරීමයි (එහි කුමන මල්වේයාර් එකතු වී ඇතිද යන්න කවුරුත් නොදන්නා). තවත් එක් පියවරක් වඩා හොඳ වන්නේ නවින්තම Linux පරිගණකයක් භාවිතා කිරීමයි, එය මම පෙන්වන්නෙමි.


ඔබට නොදන්නා දේ නිසා Linux භාවිතා කිරීමට බියක් ඇති නම්, එය ස්වාභාවිකය, නමුත් ඉගෙන ගැනීමට කාලයක් වැය කිරීමද එසේමයි. අන්තර්ජාලයේ විශාල පරිමාණයේ තොරතුරු ලබා ගත හැක. මම ඉතාමත් නිර්දේශ කරන විධාන රේඛාවේ මූලික කරුණු හඳුන්වා දෙන විශිෂ්ට කෙටි වීඩියෝවක් මෙන්න.
