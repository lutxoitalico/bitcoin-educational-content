## පෙරවදන


මෙහෙයුම් පද්ධතියක් (OS) යනු ඔබේ පරිගණකයේ සම්පත් සියල්ල කළමනාකරණය කරන ප්‍රධාන මෘදුකාංගයයි. Ubuntu වැනි විකල්ප මෙහෙයුම් පද්ධතියක් තෝරා ගැනීම ආරක්ෂාව, වියදම සහ පෞද්ගලිකත්වය සම්බන්ධයෙන් බොහෝ වාසි ලබා දිය හැක.


### Zakaj Ubuntu?




- වර්ධිත ආරක්ෂාව** : ලිනක්ස් බෙදාහැරීම් ඔවුන්ගේ ආරක්ෂාව සහ ශක්තිමත්කම සඳහා ප්‍රසිද්ධයි
- ශූන්‍ය පිරිවැය**: Ubuntu සහ බොහෝ Linux බෙදාහැරීම් නොමිලේ ලබා ගත හැක
- විශාල ප්‍රජාව**: මණ්ඩප සහ උපකාරක පඬික්කම් හරහා උපකාර කිරීමට සූදානම් පරිශීලක ප්‍රජාවක්
- ගෞරවය පෞද්ගලිකත්වය**: වැඩි විවෘතතාවය සඳහා විවෘත මූලාශ්‍ර පද්ධතියක්
- සරලකම**: පරිශීලක හිතකාමී Interface සහ භාවිතයේ පහසුව
- ධනවත් පරිසර පද්ධතිය** : විවෘත මූලාශ්‍ර මෘදුකාංග විශාල නාමාවලිය
- නිති පූර්වක සහය**: Canonical වෙතින් ආරක්ෂිත යාවත්කාලීන කිරීම්

**4.3 Ubuntu නවීකරණය කිරීම ස්ථාපනයෙන් පසු** (in French)


ඔබේ පද්ධතිය යාවත්කාලීන කිරීම යනු නව ස්ථාපනයක් කළ පසු වැදගත් පියවරක් වේ. ඔබට විකල්ප දෙකක් ඇත:


**විකල්පය 1: ග්‍රැෆිකල් පරිශීලක Interface හරහා**




- "Software and updates" යනුවෙන් යෙදුම් මෙනුවේ සොයන්න
- एप्लिकेशन स्वचालित रूपमा उपलब्ध अपडेटहरूको लागि जाँच गर्नेछ।
- ස්ථාපන යාවත්කාලීන කිරීම් සඳහා තිරයේ උපදෙස් අනුගමනය කරන්න


**විකල්පය 2: ටර්මිනල් හරහා**




- ටර්මිනල් විවෘත කරන්න (Ctrl + Alt + T)
- මෙම විධානය ටයිප් කර ලබා ගත හැකි යාවත්කාලීන කිරීම් පරීක්ෂා කරන්න:


```bash
sudo apt update
```




- ඔබට ඉල්ලා සිටින විට ඔබේ මුරපදය ඇතුළත් කරන්න
- අලුත් කිරීම් ස්ථාපනය කිරීමට, ටයිප් කරන්න :


```bash
sudo apt upgrade
```

### 5. මූලික කාර්යයන් සොයා ගැනීම


**5.1 අන්තර්ජාලය බ්‍රවුස් කිරීම


මූලිකව, ඔබට බොහෝ විට Firefox ආරම්භක තීරුවේ සොයා ගත හැක.


Firefox ආරම්භ කර බ්‍රවුසිං ආරම්භ කරන්න.


अन्य ब्राउज़रहरू (Chrome, Brave, आदि) सफ्टवेयर प्रबन्धक मार्फत वा .deb प्याकेजहरू मार्फत स्थापना गर्न सकिन्छ।


**5.2 වචන සැකසීම**


Ubuntu LibreOffice suite (Writer for word processing) සමඟ එයි.


Odpreti: Dejavnosti > Poiščite "LibreOffice Writer" ali kliknite na ikono, če se prikaže v vrstici.

**5.3 යෙදුම් ස්ථාපනය කිරීම**


සොෆ්ට්වේයර් කළමනාකරු ( "Ubuntu Software" ලෙස හැඳින්වේ): යෙදුම් සෙවීම සහ ස්ථාපනය කිරීම සඳහා දෘශ්‍ය Interface.


ටර්මිනල් වෙතින්, විධානය භාවිතා කරන්න :

```bash
sudo apt install nom-du-paquet
```


උදාහරණ:
