ඩිජිටල් යුගයේ, අපගේ දෛනික ජීවිතයේ විවිධ අංශ ආවරණය කරන මාර්ගගත ගිණුම් විශාල ප්‍රමාණයක් කළමනාකරණය කළ යුතුය, බැංකුකරණය, මූල්‍ය වේදිකා, විද්‍යුත් තැපැල්, ගොනු ගබඩා කිරීම, සෞඛ්‍යය, පරිපාලනය, සමාජ ජාල, වීඩියෝ ක්‍රීඩා ආදිය ඇතුළුව.


අපගේ මෙම ගිණුම් සෑම එකක්ම සත්‍යාපනය කිරීම සඳහා, අපි හඳුනාගැනීමේ සංකේතයක්, බොහෝ විට විද්‍යුත් තැපෑල Address, මුරපදයක් සමඟ භාවිතා කරමු. විශාල සංඛ්‍යාතයෙහි අද්විතීය මුරපද මතක තබා ගැනීමේ අසමත්‍යතාවය හමුවේ, එකම මුරපදය නැවත භාවිතා කිරීමට හෝ මතක තබා ගැනීමට පහසු කිරීම සඳහා පොදු පදනමක් ස්වල්පයක් වෙනස් කිරීමට කෙනෙකුට ප්‍රलोභනය විය හැක. කෙසේ වෙතත්, මෙම පුරුදු ඔබේ ගිණුම් ආරක්ෂාව බරපතල ලෙස අඩු කරයි.


මුරපද සඳහා අනුගමනය කළ යුතු පළමු මූලධර්මය නම් ඒවා නැවත භාවිතා නොකිරීමයි. සෑම මාර්ගගත ගිණුමක්ම අද්විතීය සහ සම්පූර්ණයෙන්ම වෙනස් මුරපදයකින් ආරක්ෂා කළ යුතුය. මෙය වැදගත් වන්නේ, ඔබේ මුරපදයක් ආක්‍රමණයකට ලක් වුවහොත්, ඔබට ඔබේ සියලුම ගිණුම් වෙත ප්‍රවේශ වීමට අවශ්‍ය නොවන නිසාය. සෑම ගිණුමක් සඳහාම අද්විතීය මුරපදයක් තිබීම සම්භාව්‍ය ප්‍රහාර වෙන් කරයි සහ ඒවායේ පරාසය සීමා කරයි. උදාහරණයක් ලෙස, ඔබේ විඩියෝ ක්‍රීඩා වේදිකාව සඳහා සහ ඔබේ විද්‍යුත් තැපෑල සඳහා එකම මුරපදය භාවිතා කරන්නේ නම්, සහ එම මුරපදය ක්‍රීඩා වේදිකාවට සම්බන්ධ මත්ස්‍ය ධාරිතාවක් හරහා ආක්‍රමණයකට ලක් වුවහොත්, ආක්‍රමණකරුට ඔබේ විද්‍යුත් තැපෑලට පහසුවෙන් ප්‍රවේශ විය හැකි අතර ඔබේ අනෙකුත් සියලුම මාර්ගගත ගිණුම් පාලනය කළ හැකිය.


දෙවන අත්‍යවශ්‍ය මූලධර්මය වන්නේ මුරපදයේ ශක්තියයි. මුරපදයක් ශක්තිමත් ලෙස සැලකෙන්නේ එය බෲට් ෆෝස් කිරීම අපහසු නම්, එනම්, පරීක්ෂණ සහ දෝෂ හරහා අනාවරණය කිරීම අපහසු නම්. මෙය ඔබේ මුරපද හැකි තාක් අහඹු, දිගු, සහ විවිධ අක්ෂර (කුඩා අකුරු, විශාල අකුරු, අංක සහ සංකේත) ඇතුළත් විය යුතු බව අර්ථ දක්වයි.


මෙම සංකේත වචන ආරක්ෂක මූලධර්ම දෙක (අද්විතීයත්වය සහ ශක්තිමත්කම) දෛනික ජීවිතයේදී අයදුම් කිරීම අපහසු විය හැක, මන්ද අපගේ සියලුම ගිණුම් සඳහා අද්විතීය, අහඹු, සහ ශක්තිමත් සංකේත වචනයක් මතක තබා ගැනීම සෑහෙන අපහසුය. මෙහිදී සංකේත වචන කළමනාකරු ක්‍රියාත්මක වේ.


මුරපද කළමනාකරු ශක්තිමත් මුරපද ජනනය කර ආරක්ෂිතව ගබඩා කරයි, ඔබට ඒවා වෙන වෙනම මතක තබා ගැනීමේ අවශ්‍යතාවයකින් තොරව ඔබේ සියලුම මාර්ගගත ගිණුම් ප්‍රවේශ වීමට ඉඩ සලසයි. ඔබට මතක තබා ගත යුතු එකම මුරපදය, මාස්ටර් මුරපදය වන අතර, එය කළමනාකරු තුළ ඔබ සුරකි ඇති සියලුම මුරපද වෙත ප්‍රවේශ වීමට ඔබට ඉඩ සලසයි. මුරපද කළමනාකරු භාවිතා කිරීමෙන් ඔබේ මාර්ගගත ආරක්ෂාව වැඩි දියුණු වේ, මන්ද එය මුරපද නැවත භාවිතා කිරීම වළක්වා, පද්ධතිගතව අහඹු මුරපද ජනනය කරයි. නමුත් එය ඔබේ සංවේදී තොරතුරු වෙත ප්‍රවේශය මධ්‍යස්ථානය කරමින් ඔබේ ගිණුම් දෛනික භාවිතය සරල කරයි.

## KeePass කෙසේ බාගත කරන්නේද?


KeePass ස්ථාපන ක්‍රියාවලිය ඔබ භාවිතා කරන මෙහෙයුම් පද්ධතිය අනුව වෙනස් වේ. Windows හෝ Linux පරිශීලකයින් සඳහා, ස්ථාපනය සරල වේ. එහෙත්, ඔබ macOS භාවිතා කරන්නේ නම්, KeePass .NET වේදිකාවේ සංවර්ධනය වීම නිසා අමතර පියවරක් අවශ්‍ය වේ, මක්නිසාද එය macOS මඟින් සෘජුවම සහය නොදක්වයි. එබැවින්, Apple උපාංගවල KeePass ක්‍රියාත්මක කිරීමට අනුකූල පරිසරයක් වින්‍යාස කිරීම අවශ්‍ය වේ.


Debian/Ubuntu uporabniki, odprite terminal in vnesite naslednje ukaze:


```bash
sudo apt-get update

sudo apt-get install keepass2

```

For Fedora:

```

sudo dnf install keepass

```

For Arch Linux:

```

sudo pacman -S keepass

```

If you are on a Windows computer, go to the [official KeePass download page](https://keepass.info/download.html), and download the latest version of the installer:
![KEEPASS](assets/notext/02.webp)
Click on the downloaded file to run it, then follow the instructions of the setup wizard to complete the installation (see next section).

For macOS users, the installation is a bit more complex. If you wish to use the original version of KeePass as on Windows, follow the instructions below. Otherwise, you can opt for [KeePassXC](https://keepassxc.org/), an alternative version compatible with macOS, which offers a slightly different interface.

To use KeePass, you will need a runtime environment for .NET applications. I recommend installing Mono for this. Go to the [official Mono page](https://www.mono-project.com/download/stable/#download-mac) in the "*macOS*" section, and click on the link to download the installation package (`.pkg`).
![KEEPASS](assets/notext/03.webp)
Open the downloaded `.pkg` file and follow the instructions to install Mono on your Mac.
![KEEPASS](assets/notext/04.webp)
Next, go to the official KeePass website and download the latest portable version in `.zip` format.
![KEEPASS](assets/notext/05.webp)
After downloading the `.zip` file, double-click to extract it. You will get a folder containing several files, including `KeePass.exe`. Open a terminal, navigate to the KeePass folder (replace `xx` with the version number):

```

cd ~/Downloads/KeePass-2.xx

```

And finally, run KeePass with Mono:

```

mono KeePass.exe

```
