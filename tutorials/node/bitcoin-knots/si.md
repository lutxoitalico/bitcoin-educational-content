### z oddaljenim Bitcoin Vozlov :


If Bitcoin Knots is installed on another machine connected to the same network, first locate the `Bitcoin.conf` file among the software files. If this file does not yet exist, you can create it. Open this file with a text editor and add the following line:


```ini
server=1
```


ගොනුව සංස්කරණය කිරීමෙන් පසු, ඔබේ මෙහෙයුම් පද්ධතිය සඳහා සුදුසු ෆෝල්ඩරය තුළ එය සුරකින්න:


|**macOS**|~/Library/Application Support/Bitcoin|
|---|---|
|**Windows**|%APPDATA%\Bitcoin|
|**Linux**|~/.Bitcoin|

මෙම මෙහෙයුම Bitcoin-QT හි Interface ග්‍රැෆික්ස් හරහා ද සිදු කළ හැක. "*Settings*" මෙනුවට යන්න, එවිට "*Options...*" තෝරන්න, සහ "*Enable RPC server*" විකල්පය සක්‍රීය කිරීම සඳහා සම්බන්ධ පෙට්ටිය සලකුණු කරන්න. `Bitcoin.conf` ගොනුව නොමැති නම්, "*Open Configuration File*" ක්ලික් කිරීමෙන් මෙම Interface සිට එය සෘජුවම සාදන්න.


![Image](assets/fr/37.webp)


ඔබේ දේශීය ජාලයේ Bitcoin Knots සත්කාරකය Address IP සොයා ගන්න. මෙය කිරීමට, [Angry IP Scanner](https://angryip.org/) වැනි මෙවලමක් භාවිතා කළ හැක. තර්කය සඳහා, ඔබේ නියුඩ් IP Address `192.168.1.18` බව අපි සම්මත කරමු.


`Bitcoin.conf` ගොනුවේ, පහත පේළි එක් කරන්න, `rpcbind=192.168.1.18` ලෙස සකසා, ඔබේ නෝඩයේ IP Address සමඟ ගැලපේ.


```ini
[main]
rpcbind=127.0.0.1
rpcbind=192.168.1.18
rpcallowip=127.0.0.1
rpcallowip=192.168.1.0/24
```


![Image](assets/fr/38.webp)


Prav tako dodajte uporabniško ime in geslo za oddaljene povezave v datoteko `Bitcoin.conf`. Poskrbite, da zamenjate `loic` z vašim uporabniškim imenom in `my_password` z močnim geslom:


```ini
rpcuser=loic
rpcpassword=my_password
```
