- Git භාවිතා කර ස්ථානීයව දායක වන්න**: ඔබ PlanB ජාලයට නිතර හෝ වැදගත් දායකත්වයන් ලබා දීමට සැලසුම් කරන්නේ නම්, මෙම ක්‍රමය වඩාත් සුදුසු වේ. ඔබේ පරිගණකයේ ස්ථානීය Git පරිසරය පිහිටුවීම මුලදී සංකීර්ණ ලෙස පෙනෙන්නට ඇති නමුත්, දිගු කාලීනව මෙම ආකාරය වඩා කාර්යක්ෂම වේ. එය වෙනස්කම් වඩාත් ව්‍යවස්ථානුකූලව කළමනාකරණය කිරීමට ඉඩ සලසයි. ඔබට මෙය නවකමක් නම්, කනගාටු නොවන්න, **අපි මෙම උපකාරකය තුළ ඔබේ පරිසරය පිහිටුවීමේ සම්පූර්ණ ක්‍රියාවලිය පැහැදිලි කරමු** (වචන රේඛා කිසිවක් ටයිප් කිරීමට ඔබට අවශ්‍ය නොවේ කියලා පොරොන්දු වෙනවා ^^).

උදාහරණයක් ලෙස, නව උපකාරක පංතියක් නිර්මාණය කරන විට, ඔබේ උපකාරක පංතිය සඳහා ටැගයන්, එහි මාතෘකාව, සහ ඔබේ ගුරු ID ඇතුළත් කිරීමට YAML (`.yml`) ගොනුවක් නිර්මාණය කළ යුතු වේ. Obsidian මෙම වර්ගයේ ගොනු සංස්කරණය කිරීමේ හැකියාව ලබා නොදෙයි, එබැවින් ඔබට කේත සංස්කාරකයක් අවශ්‍ය වේ.
